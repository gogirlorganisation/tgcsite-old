Welcome to The Girl Code

Mission

At The Girl Code, we aim to bridge the gender gap in the tech community by inspiring young girls to learn programming by hosting workshops at schools and universities local to them. Through our platform and intuitive curriculum, we plan to give rise to a new generation of female programmers set to take the world by storm.

Introducing #CodewithConfidence

#CodewithConfidence

The Girl Code launched a video campaign titled, "Code with Confidence," wherein female programmers from around the world shared their programming journey in order to inspire young girls. Visit @thegirlcode2018 to watch it, and start your programming journey today!

Confidence

Curiosity

The Girl Code endeavours to arouse Curiosity in our students, the curiosity that is essential to pursue Computer Science. Curiosity is an essential tool to motivate and inspire them.

Community

The Girl Code seeks to build a tight-knit Community of young female programmers, always there to support, motivate, and inspire each other.
